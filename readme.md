## Elementos  de un proyecto 🚀

_Estas instrucciones te permitirán obtener una copia del proyecto en funcionamiento en tu máquina local para propósitos de desarrollo y pruebas._

**package.json** nos muetra todo lo importante de la app
Campos:
- name: nombre app
- version: de la app
- dependences: lo que necesita para que funcione el proyecto
- scripts: comandos disponibles para ejecutar

**node_modules** Almacena todas las dependencias de nuestro proyecto

### Creación de nuestro proyecto ⚙️

```
npx create-react-app nombre_app
```


### Transpilación 🔧

_Cuando nuestra aplicación react se encuentra ejecutando y nosotros ejecutamos cambios, estos se ven afectados dentro de un proceso de transpilación_

_La transpilación es cuando nuestro servidor se encuentra ejecutándose y se encuentran cambios no reflejados. React los toma y los actualiza para que tomen parte dentro de los cambios en nuestro servidor corriendo_
